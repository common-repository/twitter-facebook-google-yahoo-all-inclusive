All Inclusive plugin allows to your visitors to register, login and comment with their Facebook, Twitter, Google, Yahoo with their WordPress account.

== Description ==

All Inclusive adds social  buttons on the login, register and comment forms of your WordPress Website.

The buttons offerirg login and registration using a Facebook, Twitter, Google, Yahoo and WordPress account.
Make your website easy for new members to register with your website and existing members to login.


== Installation ==

1. Upload the plugin to the "/wp-content/plugins/" directory.
2. Activate it in the "Plugins" admin panel.
3. Go to "Settings | All Inclusive" administration page to configure social media providers. 


If you like to add the All Inclusive login or registration forms to another location in your theme, you can insert the following code in that location:

`<?php do_action( 'all_inclusive_form' ); ?>`
